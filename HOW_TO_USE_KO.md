# 100칸 계산 생성기 - 사용 가이드

구매해 주셔서 감사합니다!

## 📖 앱 정보

100칸 계산 생성기는 덧셈, 뺄셈, 곱셈, 나눗셈 문제를 자동으로 생성하는 웹 애플리케이션입니다. **인쇄**하거나 **온라인 입력 모드**로 연습할 수 있습니다.

### ✨ 주요 기능

- **2가지 모드**: 인쇄 모드(종이 연습), 입력 모드(온라인 연습)
- **4가지 연산**: 덧셈, 뺄셈, 곱셈, 나눗셈
- **유연한 격자 크기**: 행과 열을 독립적으로 설정(3-15), 정사각형 또는 직사각형
- **16개 언어**: 일본어, 영어, 중국어(간체/번체), 한국어, 태국어, 인도네시아어, 베트남어, 힌디어, 아랍어, 러시아어, 네덜란드어, 스페인어, 프랑스어, 독일어, 포르투갈어
- **고급 기능**: 음수, 세로 형식, 고정 숫자, 무작위 배열
- **입력 모드 전용**: 타이머, 자동 채점, 성적 기록, 최고 기록 표시
- **설정 자동 저장**: 이전 설정 기억

## 🚀 빠른 시작

1. ZIP 파일 압축 해제, index.html 열기
2. 언어 및 연산 유형 선택
3. 행×열 설정(예: 10×10 또는 5×10)
4. "생성" 버튼 클릭
5. 모드 선택: 인쇄 또는 입력

## 💻 입력 모드

- 타이머 시작
- 답안 입력(키보드 또는 터치 숫자 키패드)
- "채점" 클릭하여 결과 확인
- 성적 자동 기록(최근 10회)

## 🎨 고급 기능

- **세로 형식**: 숫자 위치 바꾸기
- **음수**: -999~999 범위
- **고정 숫자**: 특정 숫자 지정 연습
- **무작위 배열**: 매번 다른 순서

## 💾 설정 자동 저장

문제 생성 시 모든 설정이 자동 저장되며, 다음 실행 시 복원됩니다.
