# 百格計算生成器 - 用戶指南

感謝您的購買！

## 📖 關於此應用

百格計算生成器是一個自動創建加減乘除題目的Web應用程式。您可以**列印**或使用**線上輸入模式**練習。

### ✨ 主要功能

- **2種模式**：列印模式（紙質練習）、輸入模式（線上練習）
- **4種運算**：加法、減法、乘法、除法
- **靈活的網格大小**：獨立設定行數和列數（3-15），正方形或長方形
- **16種語言**：日語、英語、簡體中文、繁體中文、韓語、泰語、印度尼西亞語、越南語、印地語、阿拉伯語、俄語、荷蘭語、西班牙語、法語、德語、葡萄牙語
- **進階功能**：負數、直式格式、固定數字、隨機排列
- **輸入模式專屬**：計時器、自動評分、成績記錄、最佳時間顯示
- **自動儲存設定**：記住您之前的配置

## 🚀 快速開始

1. 解壓ZIP檔案，開啟index.html
2. 選擇語言和運算類型
3. 設定行數×列數（例：10×10或5×10）
4. 點擊「生成」按鈕
5. 選擇模式：列印或輸入

## 💻 輸入模式

- 啟動計時器
- 輸入答案（鍵盤或觸控數字鍵盤）
- 點擊「評分」查看結果
- 成績自動記錄（最近10次）

## 🎨 進階功能

- **直式格式**：交換數字位置
- **負數**：-999到999範圍
- **固定數字**：指定特定數字練習
- **隨機排列**：每次順序不同

## 💾 設定自動儲存

生成題目時自動儲存所有設定，下次啟動時恢復。
