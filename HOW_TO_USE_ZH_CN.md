# 百格计算生成器 - 用户指南

感谢您的购买！

## 📖 关于此应用

百格计算生成器是一个自动创建加减乘除题目的Web应用程序。您可以**打印**或使用**在线输入模式**练习。

### ✨ 主要功能

- **2种模式**：打印模式（纸质练习）、输入模式（在线练习）
- **4种运算**：加法、减法、乘法、除法
- **灵活的网格大小**：独立设置行数和列数（3-15），正方形或长方形
- **16种语言**：日语、英语、简体中文、繁体中文、韩语、泰语、印度尼西亚语、越南语、印地语、阿拉伯语、俄语、荷兰语、西班牙语、法语、德语、葡萄牙语
- **高级功能**：负数、竖式格式、固定数字、随机排列
- **输入模式专属**：计时器、自动评分、成绩记录、最佳时间显示
- **自动保存设置**：记住您之前的配置

## 🚀 快速开始（3步）

### 步骤1：打开文件

1. 解压下载的ZIP文件
2. 双击文件夹中的"**index.html**"
3. 在浏览器中打开（Chrome、Safari、Edge等）

**💡 提示**：
- **无需互联网连接**（离线使用）
- 保存到桌面以便访问

---

### 步骤2：创建题目

#### 基本设置
1. **选择语言**（页面顶部下拉菜单）
2. **选择运算类型**（加法/减法/乘法/除法）
3. **设置行数（垂直）**和**列数（水平）**
   - 示例：10×10（正方形）、5×10（长方形）
4. **设置数字范围**
5. 点击"**生成**"按钮

#### 选择模式

**🖨 打印模式**：
- 显示题目和答案表
- 点击"打印"在A4纸上打印
- 对折纸张使用

**💻 输入模式**：
- 直接在屏幕上输入答案
- 计时器追踪速度
- 自动评分提供即时反馈
- 记录成绩历史

---

## 📐 网格大小设置

### 正方形（传统）
```
行数：10
列数：10
结果：10×10正方形
```

### 长方形（新功能！）
```
行数：5
列数：10
结果：5×10长方形（横向）
```

**应用场景**：
- 小型（5×8）：初学者、短时练习
- 标准（10×10）：经典百格
- 横向（8×15）：乘法表练习
- 纵向（15×5）：特定数字练习

## 💻 使用输入模式

### 基本操作

1. 生成后，点击"**输入模式**"标签
2. **启动计时器**
3. 输入答案（键盘或数字键盘）
4. 完成后，点击"**评分**"按钮

### 输入方法

#### PC（键盘）
- **输入数字**：数字键盘或数字键
- **移动到下一格**：Enter、Tab、方向键
- **负号**：`-`键

#### 平板/手机
- 右侧**屏幕数字键盘**输入
- **大按钮**易于触摸
- **建议横屏**（自动分屏）

### 移动方向

选择"移动方向"：
- **向右移动**：水平方向（默认）
- **向下移动**：垂直方向

### 计时器功能

- **开始**：开始计时
- **暂停**：暂停计时（可恢复）
- **重置**：重置为0

**精度**：10毫秒（00:00.00格式）

### 评分功能

点击"评分"按钮查看：
- ✅ **正确**：绿色显示
- ❌ **错误**：红色显示
- 📊 **统计**：正确数、错误数、准确率、时间

### 成绩记录

- **自动保存**：记录最近10次
- **显示**：日期/时间、正确数、准确率、时间
- **最佳时间**：最快记录显示金色"BEST"徽章

---

## 🎨 高级功能

### 1. 竖式格式

交换数字位置，适合竖式计算。

### 2. 包含负数

创建带负数的题目。
- 范围扩展至-999到999
- 示例：-5 + 3 = -2

**子功能 - 随机负数化**：
- 约50%概率转换为负数

### 3. 固定特定数字

指定使用的数字。

**用法**：
```
✅ 勾选"固定特定数字"
固定数字（行）：1,2,3,4,5
固定数字（列）：7
```
→ 仅练习7的乘法表

**子功能 - 随机排列**：
- 随机打乱固定数字
- 每次顺序不同

---

## 💾 自动保存设置

**生成题目时，以下内容自动保存**：
- 运算类型
- 行数和列数
- 数字范围（最小/最大）
- 复选框状态（竖式格式、负数等）
- 固定数字内容

**下次启动**：从之前的设置开始
